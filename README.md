# Module 1: The AI-Data Analyst

Welcome to your personal repository for Module 1 of Climate Solutions 208!

## What This Repo Is For

This is where you'll complete all in-class exercises and push your work for Sessions 1-4. Each session builds on the previous one as we work toward building an interactive emissions dashboard.

## Getting Started

1. **Clone this repository** to your local machine
2. **Open it in VS Code** (or Antigravity IDE)  
3. **Set up your environment** — ask your AI assistant to help!
4. **Follow the session guides** on the course website

## Sessions

| Session | Topic | Activity |
|---------|-------|----------|
| 1 | IDE Setup & GitHub | Setting up your development environment |
| 2 | Hello Climate | First data analysis with OWID CO2 data |
| 3 | Data Cleaning | Spotting and fixing data quality issues |
| 4 | Dashboards | Building interactive web visualizations |

## Repository Structure

```

└── outputs/        # Generated charts and outputs
```
